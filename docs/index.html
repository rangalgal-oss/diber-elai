פרטי לחלוטין

התמלול קורה על המחשב שלך. שום אודיו ושום טקסט לא נשלחים לאינטרנט — מושלם למסמכים רגישים.

עברית מדויקת

מבוסס על מודל עברית ייעודי (ivrit.ai) — מבין ניבים, שמות וניסוח טבעי הרבה יותר טוב מהכתבה גנרית.

עברית ואנגלית יחד

אומרים "תפתח את ה‑browser" — והמילים באנגלית נשארות באנגלית. בלי תעתיק מביך.

נכנס לכל תוכנה

הטקסט מודבק אוטומטית לאן שהסמן נמצא — וורד, וואטסאפ, מייל, דפדפן, כל מקום.

קליק או קיצור

החזקת גלגלת העכבר להקלטה, או קיצור מקלדת גלובלי. עובד מכל מקום במחשב.

היסטוריה נשמרת

כל מה שהכתבת נשמר בחלון — להעתיק שוב מתי שרוצים, גם אחרי שסוגרים את התוכנה.

התקנה

שלושה צעדים, פחות מדקה

בלי הרשאות מנהל, בלי הגדרות. מתקינים ומתחילים לדבר.

1

מורידים ומריצים

לחצו על ההורדה ופתחו את הקובץ DiberElai-Setup שירד.

2

מתקינים

עוקבים אחרי האשף (אפשר לסמן קיצור לשולחן העבודה והפעלה עם Windows). ההתקנה אישית — בלי UAC.

3

מדברים

בועה קטנה תופיע בפינה. מחזיקים את גלגלת העכבר או Ctrl+Space, מדברים — והטקסט נכנס.

אם Windows מציג חלון כחול "Windows protected your PC"

זה תקין — התוכנה עדיין לא חתומה דיגיטלית, אז Windows זהיר. לחצו על More info ואז על Run anyway, וההתקנה תמשיך כרגיל. (אם האנטי‑וירוס מתלונן — אשרו; התוכנה מאזינה למקלדת כדי לאפשר את קיצור ההקלטה, ולכן לפעמים מסומנת בטעות.)

איך משתמשים

הכל בהישג יד אחת

להקליט

החזיקו את גלגלת העכבר (הלחצן האמצעי) ודברו; שחררו והטקסט נכנס. או לחצו Ctrl+Space.

להזיז את הבועה

גוררים את הבועה לאן שנוח. קליק ימני עליה פותח את חלון ההיסטוריה וההגדרות.

להתאים אישית

בהגדרות אפשר לבחור גודל בועה, שקיפות, מודל וקיצור מקלדת. החלון ניתן למתיחה כרצונכם.

רץ ברקע

התוכנה יושבת באייקון ליד השעון. משם פותחים את החלון או יוצאים לגמרי.
